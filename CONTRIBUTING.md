# Contributing

Terima kasih sudah tertarik berkontribusi ke TodoFlow.

## Local Setup

1. Pastikan JDK 21 tersedia.
2. Jalankan `./mvnw test` atau `.\mvnw.cmd test` di Windows.
3. Jalankan aplikasi dengan `./mvnw spring-boot:run`.
4. Buka `http://localhost:8080`.

## Pull Request

- Buat branch dengan nama yang menjelaskan perubahan.
- Tambahkan atau perbarui test untuk perubahan perilaku.
- Pastikan seluruh test lulus.
- Jelaskan alasan dan dampak perubahan di pull request.
